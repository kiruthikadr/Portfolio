Event Management System

A full-stack Event Organizing web application designed to streamline event planning and management. The backend is built using Spring Boot (Java) with Maven for dependency management and MySQL as the relational database. It provides robust APIs for user authentication, event scheduling, ticket booking, and real-time notifications.

The frontend is developed using Node.js and modern JavaScript frameworks, ensuring a responsive and interactive user experience. The platform supports both Admin and User roles, enabling seamless coordination of events, attendees, and resources.

Key features include role-based access control, secure data persistence, RESTful API integration, automated email confirmations, and scalable architecture to support large event operations.
